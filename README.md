# Портфолио-проект: Лендинг магазина аксессуаров

Этот проект — пример создания современного, адаптивного лендинга с использованием React, Vite и Tailwind CSS.

## 🚀 Технологии

- **React** — библиотека для создания пользовательских интерфейсов
- **Vite** — быстрый инструмент сборки
- **Tailwind CSS** — utility-first CSS framework

## 📦 Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите проект в режиме разработки:
```bash
npm run dev
```

3. Откройте [http://localhost:5173](http://localhost:5173) в браузере

## 🏗️ Структура проекта


## 📋 Основные компоненты

- **Header** — навигация сайта
- **Hero** — главный баннер с призывом к действию
- **ProductCard** — карточка товара
- **Footer** — подвал сайта
- **Home** — главная страница с товарами

## 🎨 Стиль

Проект выполнен в минималистичном стиле с использованием:
- Пастельных тонов
- Современной типографики (Poppins)
- Адаптивной верстки

## 🌐 Деплой на Vercel

1. Создайте аккаунт на [vercel.com](https://vercel.com)
2. Подключите GitHub-репозиторий проекта
3. Нажмите 'Deploy' и получите ссылку на свой лендинг

## 📝 Команды

- `npm run dev` — запуск в режиме разработки
- `npm run build` — сборка для продакшена
- `npm run preview` — предпросмотр собранного проекта
